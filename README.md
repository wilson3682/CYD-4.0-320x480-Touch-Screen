# CYD-4.0-320x480-Touch-Screen
Files for testing CYD 4.0'' 320x480 TFT ESP32-32E ESP32 Display with ST7796S Driver.
#
### CYD Board used: 
- https://www.amazon.com/dp/B0FGJJ24S1?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1
###
### Important Information Links: 
- Board support files: https://hosyond.com/
- https://www.lcdwiki.com/4.0inch_ESP32-32E_Display#Package_Library
- https://www.youtube.com/watch?v=agfuTNZGGl4&t=11s
- https://resinchemtech.blogspot.com/2025/08/cyd.html
#
### Library used: 
- https://github.com/Bodmer/TFT_eSPI
#
# How to use:
- Rename the User_Setup.h on Documents/Arduino/libraries/TFT_eSPI library folder to something like User_Setup.h_Original.
- Then copy the modified User_Setup.h to the same TFT_eSPI folder.
- Run the sample sketch using Arduino IDE 1.8.19
- Esp32 dev version 3.3.2
